RULES Player

* This documentation is not finished yet.

RULES has an image sequence WEB Player with painting and commenting functions. It can operate with JPEG and PNG images only. But for as for CG, usually only JPEGs are used for preview (movies, dailies) due size.

Settings

Pre Create

All images will be created in memory before draw. In this case, playback will be more smooth. But needed much more RAM to keep not compresses images. It is very needed parameter for a constant FPS. By default it is OFF - not to save your phone or tablet.

WEB GL

WEB GL allows more smooth playback. Need more RAM, especially with PreCreate. By default it is ON, but if your browser does not supports it, it will be turned off automatically.

RULES Player Online Examples

All examples will be opened in a new tab (window). With PreCreate (see above) and WEB GL they should be played smoothly at 24 FPS.

Note! This site is on the cheapest hosting tariff. And sometimes it hangs for a seconds or gives corrupted images.
