Rendring Flags Window:

setted flags on scanned objects are represented by colors:
"all ON" - all flags are setted on,
"some ON some OFF" - not all flags have the same values,
"all OFF " - all flags are setted on,

"Set" buttons set on selected hierarchy only current flag on or off, depending on its checkbox value. This command works faster then set all command.
"sAll" button sets all rendering flags according to their checkboxes values.
"refresh" button scans the selected hierarchy and displays flags information by colors.

Output field displays status.

"help" button opens this HTML page.

During long scanning on heavy hierarchy you will see "scanning..." caption. Number of points represents thousands of scanned shapes.
