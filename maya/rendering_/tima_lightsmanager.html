Lights Manager:

Для установки атрибутов сразу на многих источниках света. Оперирует со всеми выделеными источниками света с учетом иерархи. Окно показывает информацию о первом попавшимся ему источнике света из выделенных.

 

"set" - set attribute fo current lights.
"Light" - light color.
"Shad" - shadow color.

mental ray:
"Clear map name;" - set smapFilename attribute to "".
"Set shape name - " set shadow file name to the name of each lightShape name.
"skip empty maps" - skip light if it smapFilename == "".
"prefix string A:" - set smapFilename = stringA + smapFilename.
"suffix string A:" - set smapFilename = smapFilename + stringA.
"A:" - stringA input field.
"replace string A with string B:" - search stringA in smapFilename and replace it with stringB.
"B:" - stringB input field.
"refresh" - refresh window information.
"print in script editor" - output setAttr commands in Script Editor.








Output field displays status.
