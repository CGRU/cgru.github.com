Unfeeze Translation.

You can restore object translation according to it's pivot or bounding box.

Options Window:

 

BBMin - the minimun point of objects's bounding box along the axis.
BBMid - average between minimum and maximum.
BBMax - the maximum.
Unfeeze - main action.
Translate - translates object pivot.

Also you can freeze object translation:
all - translation, rotation and scale.
translation - translation only.
rotation - rotation only.
scale - scale only.

If you does not accept pivot position, you can translate if according to the object bounding box.
