Sets Editor.

Sets editor window allows boolean operations between many sets or sets and selected components(objets). You can place the result of operation in new set, one of operated set or selection (choose it in "Place Result In" frame).

 

Sets List Manager:

"clear" - clear the whole list.
"add from selection" - add to the list all sets, witch connected to selection.

operatrions:
doubleclick:
main sets list - select set members.
set B list - push set to list A.
set A list - select current set node.
RMB - list a popup to rename selected set node.
delete - delete set from list.
drag&drop - move sets between lists.

aux operations:
"sel set" - select set node.
"sel mem" - select members of all selected in editor window sets.
"add sel" - add selection to every selected in editor sets.
"help" - open this HTML page.
